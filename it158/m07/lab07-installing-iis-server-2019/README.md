# Lab 07: Installing IIS on Windows Server 2019

**Module:** M07 | **Course:** IT158 | **Date:** October 9, 2025

## Tools Used
- Windows Server 2019
- Internet Information Services (IIS)
- Server Manager
- Oracle VirtualBox

## What I Did
Installed and configured Internet Information Services (IIS) on
Windows Server 2019 using Server Manager. Added the IIS role,
installed required features including .NET Framework 4.7, and
verified web server functionality by confirming IIS was serving
pages correctly after installation.

## Key Concepts
- IIS role installation via Server Manager
- .NET Framework feature installation
- Web server configuration on Windows Server
- IIS Manager interface
- Verifying web server functionality

## Why This Matters
IIS is Microsoft's web server platform used across enterprise
environments to host internal and external web applications.
Installing and configuring IIS is a core Windows Server
administration skill tested on Server+ and required for
system administrator and web operations roles.
