# Lab 16: Using the Linux tar Command for Data Backup

**Module:** M16 | **Course:** IT158 | **Date:** November 2025

## Tools Used
- Linux Server
- tar command
- NDG NetLab

## What I Did
Used the Linux tar command to back up data on a Linux file system.
Created compressed archive files using tar, extracted archived data,
and compared the Linux tar backup approach to Windows Server Backup
methods to understand the differences between the two platforms.

## Key Concepts
- tar command syntax and options
- Creating compressed archives with tar
- Extracting data from tar archives
- Linux backup methodology vs Windows backup
- Command line backup automation on Linux

## Why This Matters
The tar command is the standard backup and archiving tool on Linux
and is used across every Linux server environment in the world.
Understanding how to back up and restore data using tar is essential
for Linux administrators, DevOps engineers, and cloud operations
roles. It is also directly comparable to Windows Server Backup
showing cross-platform server administration competency.
