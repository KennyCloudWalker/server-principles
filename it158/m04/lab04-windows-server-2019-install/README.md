# Lab 04: Windows Server 2019 Installation

**Module:** M04 | **Course:** IT158 | **Date:** September 20, 2025

## Tools Used
- Oracle VirtualBox
- Windows Server 2019
- NDG NetLab

## What I Did
Installed and configured two Windows Server 2019 virtual machines.
Configured one server with a static IP address and the other to obtain
its IP through DHCP. Both servers were placed on a NAT network to
simulate an enterprise environment. Verified connectivity and confirmed
both servers were operational after installation.

## Key Concepts
- Windows Server 2019 installation process
- Static vs DHCP IP configuration on a server
- NAT network configuration in VirtualBox
- Post-installation verification and connectivity testing
- Virtual machine setup for server environments

## Why This Matters
Installing and configuring Windows Server is the foundational skill
for any server administrator. Every enterprise runs Windows Server
for Active Directory, DNS, file sharing, and more. Knowing how to
deploy and configure servers from scratch is a core Server+ competency
and daily task for system administrators.
