# Lab 05: Linux VM Installation

**Module:** M05 | **Course:** IT158 | **Date:** September 26, 2025

## Tools Used
- Oracle VirtualBox
- Ubuntu Server
- NDG NetLab

## What I Did
Installed and configured a Linux server using Ubuntu Server in
VirtualBox. Configured a static IP address per the lab network
addressing table, verified connectivity after installation, and
documented the full installation and configuration process.

## Key Concepts
- Ubuntu Server installation process
- Static IP configuration on Linux
- VirtualBox VM setup for Linux environments
- Post-installation connectivity verification
- Linux vs Windows Server differences

## Why This Matters
Linux powers the majority of web servers, cloud infrastructure, and
enterprise back-end systems worldwide. Being comfortable installing
and configuring Linux servers is essential for system administrators,
cloud engineers, and security professionals. This is directly
applicable to AWS, Azure, and GCP cloud roles.
