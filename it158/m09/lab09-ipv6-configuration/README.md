# Lab 09: IPv6 Configuration

**Module:** M09 | **Course:** IT158 | **Date:** October 2025

## Tools Used
- Windows Server 2019
- Linux Server
- NDG NetLab

## What I Did
Installed and configured IPv6 on both Windows Server and Linux
server environments. Set up IPv6 addressing, verified connectivity
between hosts using IPv6, and documented the configuration process
for both operating systems.

## Key Concepts
- IPv6 address structure and configuration
- IPv6 setup on Windows Server 2019
- IPv6 setup on Linux Server
- Dual-stack IPv4 and IPv6 environments
- IPv6 connectivity verification

## Why This Matters
IPv6 is the future of networking and is increasingly required in
enterprise environments as IPv4 addresses are exhausted. Being
able to configure IPv6 on both Windows and Linux servers is a
core Server+ competency and directly applicable to network
and system administrator roles.
