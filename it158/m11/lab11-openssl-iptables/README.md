# Lab 11: OpenSSL Encryption and IPtables Firewalling on Linux

**Module:** M11 | **Course:** IT158 | **Date:** October 2025

## Tools Used
- Linux Server
- OpenSSL
- IPtables
- NDG NetLab

## What I Did
Set up OpenSSL for encryption on a Linux server and configured
IPtables firewall rules to control network traffic. Practiced
generating certificates and keys with OpenSSL and applied
IPtables rules to allow and deny specific traffic on the server.

## Key Concepts
- OpenSSL certificate and key generation
- Symmetric and asymmetric encryption basics
- IPtables firewall rule configuration
- Allowing and denying traffic with IPtables
- Linux server security hardening

## Why This Matters
OpenSSL and IPtables are two of the most important security tools
on any Linux server. OpenSSL powers HTTPS encryption across the
internet. IPtables is the foundation of Linux firewall management.
Knowing how to use both is essential for Linux administrators,
security engineers, and cloud operations roles.
