# Lab 13: Configuring the Volume Shadow Service

**Module:** M13 | **Course:** IT158 | **Date:** November 2025

## Tools Used
- Windows Server 2019
- Volume Shadow Copy Service (VSS)
- NDG NetLab

## What I Did
Configured the Volume Shadow Copy Service on Windows Server 2019.
Set up shadow copies on server volumes, configured schedules for
automatic shadow copy creation, and verified that shadow copies
were being created and stored correctly for recovery purposes.

## Key Concepts
- Volume Shadow Copy Service (VSS) overview
- Configuring shadow copies on server volumes
- Scheduling automatic shadow copy creation
- Shadow copy storage management
- Using VSS for point-in-time recovery

## Why This Matters
Volume Shadow Copy is how Windows Server protects data by creating
point-in-time snapshots of volumes. When files are accidentally
deleted or corrupted, VSS allows recovery without a full backup
restore. This is a core Server+ topic and an essential skill for
any Windows Server administrator managing production environments.
