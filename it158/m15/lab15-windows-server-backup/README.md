# Lab 15: Configuring and Using Windows Server Backup

**Module:** M15 | **Course:** IT158 | **Date:** November 2025

## Tools Used
- Windows Server 2019
- Windows Server Backup
- NDG NetLab

## What I Did
Configured and used Windows Server Backup to back up the entire
system and specific sets of files and folders on Windows Server
2019. Practiced scheduling backups, selecting backup targets,
and verifying successful backup completion.

## Key Concepts
- Windows Server Backup installation and configuration
- Full system backup vs file and folder backup
- Backup scheduling and automation
- Backup target selection and storage
- Verifying backup job completion

## Why This Matters
Backup and recovery is one of the most critical responsibilities
of any server administrator. Without reliable backups, a single
hardware failure or ransomware attack can destroy an organization.
Knowing how to configure and verify Windows Server Backup is a
core Server+ competency and essential for any system administrator
role managing production servers.
