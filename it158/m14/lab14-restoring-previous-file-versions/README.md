# Lab 14: Restoring Previous Versions of Files

**Module:** M14 | **Course:** IT158 | **Date:** November 2025

## Tools Used
- Windows Server 2019
- Volume Shadow Copy Service (VSS)
- NDG NetLab

## What I Did
Used the Volume Shadow Copy Service to restore previous versions
of files on Windows Server 2019. Practiced recovering files that
had been modified or deleted by accessing shadow copies and
restoring them to their previous state.

## Key Concepts
- File recovery using Volume Shadow Copy
- Accessing previous versions through Windows Explorer
- Restoring individual files vs full volume restore
- Shadow copy as a recovery mechanism
- Data protection best practices on Windows Server

## Why This Matters
Accidental file deletion and corruption happen in every
organization. Being able to quickly restore a previous version
of a file without a full backup restore saves time and prevents
data loss. This skill is directly applicable to system
administrator, help desk, and server operations roles and is
a core Server+ competency.
