# Lab 08: Monitoring and Managing Disk Space in Windows Server 2019

**Module:** M08 | **Course:** IT158 | **Date:** October 2025

## Tools Used
- Windows Server 2019
- Disk Management
- NDG NetLab

## What I Did
Learned the steps and process for monitoring and managing disk space
on Windows Server 2019. Used built-in Windows Server tools to monitor
disk usage, manage volumes, and maintain storage health on a live
server environment.

## Key Concepts
- Windows Server disk management tools
- Volume monitoring and management
- Disk space analysis and maintenance
- Storage health monitoring
- Server storage best practices

## Why This Matters
Disk space management is a critical daily task for server
administrators. A full disk can crash a server, corrupt data, and
take down critical services. Knowing how to monitor and manage
storage proactively is a core Server+ competency and essential
for any system administrator role.
