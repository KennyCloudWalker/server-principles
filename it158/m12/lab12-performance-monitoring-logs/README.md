# Lab 12: Performance Monitoring and Log Retrieval

**Module:** M12 | **Course:** IT158 | **Date:** November 2025

## Tools Used
- Windows Server 2019
- Linux Server
- NDG NetLab

## What I Did
Monitored performance metrics and retrieved logs on both Windows
Server 2019 and Linux server environments. Used built-in tools to
analyze CPU, memory, and disk performance, and practiced collecting
and reviewing system logs for troubleshooting and auditing purposes.

## Key Concepts
- Performance monitoring on Windows Server 2019
- Performance monitoring on Linux Server
- System log retrieval and analysis
- CPU, memory, and disk metric monitoring
- Log review for troubleshooting and auditing

## Why This Matters
Performance monitoring and log analysis are daily tasks for any
server administrator. Knowing when a server is under stress before
it fails prevents outages. Log retrieval is also a core skill for
incident response — logs are evidence. This is tested on Server+
and directly applicable to system administrator and SOC roles.
