# Lab 06: Server Roles — DNS, DHCP, and Active Directory

**Module:** M06 | **Course:** IT158 | **Date:** October 5, 2025

## Tools Used
- Windows Server 2019
- Server Manager
- NDG NetLab

## What I Did
Deployed and integrated DNS, DHCP, and Active Directory Domain
Services (AD DS) roles on Windows Server 2019. Configured each
role, verified domain functionality, and demonstrated how these
three core services work together in an enterprise environment.

## Key Concepts
- DNS role installation and configuration
- DHCP role setup and scope configuration
- Active Directory Domain Services deployment
- Domain controller promotion
- Integration of DNS, DHCP, and AD DS

## Why This Matters
DNS, DHCP, and Active Directory are the three pillars of every
Windows enterprise network. Every organization running Windows
depends on these services. Deploying and configuring them is
a core Server+ and system administrator competency.
