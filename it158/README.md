# IT158 — Server+ (Server Principles)

**Institution:** Coastline Community College
**Term:** Fall 2025
**Instructor:** Dr. Brown
**Certification Alignment:** CompTIA Server+
**Lab Environment:** NDG NetLab, VirtualBox, Windows Server 2019, Ubuntu Server

---

## Course Overview

Hands-on server administration labs covering Windows Server 2019 and
Linux. Topics include installation, server roles, IIS, DNS, DHCP,
Active Directory, IPv6, NAT, OpenSSL, disk management, performance
monitoring, backup and recovery.
